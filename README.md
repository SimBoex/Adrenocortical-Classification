# PredictiveModels_AdrenocorticalTumors

1. In the current directory:
    - 1_ stands for the approaches using only radiomics for the binary classification


## To run the code:
1. Create a conda environment: 
    
    conda create -n PredictiveModelsAdrenoTumors  python=3.10.3 anaconda

2. conda activate PredictiveModelsAdrenoTumors  

3. Install the dependencies: 

    pip install -r requirements.txt
